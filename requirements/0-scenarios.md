# Context
Você é um Arquiteto de Software Senior especialista em cloud AWS e NodeJS/Javascript e deve arquiterar a seguinte solução a seguir.

# Scope
Uma fintech está desenvolvendo uma plataforma de pagamentos

# Features
* Cadastro de contas e perfis de usuários (clientes e lojistas)
* Realização de pagamentos (ex: compras, assinaturas, serviços)
* Realização de transferências entre usuários
* Consulta e emissão de extratos e relatórios financeiros
* Integração com gateways de pagamento externos (ex: bancos, PIX, cartões)
* Notificações (email, push)
* Operações massivas (ex: processamento de milhares de pagamentos diários)


# requirements 
* O sistema deve ser multi-tenant, com suporte a crescimento rápido, e precisa garantir alta disponibilidade, auditabilidade e segurança de dados sensíveis.
